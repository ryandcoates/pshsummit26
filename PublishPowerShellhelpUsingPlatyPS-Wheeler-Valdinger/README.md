# Publish PowerShell help content using PlatyPS

PlatyPS 1.0 GA is out. Learn how to use this new version of PlatyPS to create new documentation,
update existing documentation, and publish your docs to a static website.

## Resources

Presentation content

- [Unlocking the secrets of PowerShell Help](https://seanonit.org/docs/powershell-help/)
- [PlatyPS v1.0](https://seanonit.org/docs/platyps/)
- [GitHub - steviecoaster/PlatyPS.Hosting module](https://github.com/steviecoaster/PlatyPS.Hosting)
  - PlatyPS.Hosting is a PowerShell module that extends the Microsoft.PowerShell.PlatyPS v1 help
    authoring workflow. It provides three commands that together let you go from a loaded module to
    a published, browser-friendly HTML documentation site hosted on IIS.
