# PowerShell Style Guide for Coding Agents + Humans

Lightning talk materials from the PowerShell + DevOps Global Summit 2026.

## Session Overview

This 5-minute lightning talk covers how a comprehensive PowerShell style guide — originally written for humans — became the leverage point for an autonomous AI coding workflow. The lightning talk walks through:

- The "20x Challenge" inspired by Jeffrey Snover's Summit 2025 keynote
- Building a ~115 KB style guide by feeding representative scripts to a "council of LLMs" for forensic code analysis
- Discovering GitHub Copilot agent instructions and getting AI to actually follow the guide
- An autonomous dev loop where GitHub Copilot coding agent writes PRs, Claude Code orchestrates code review, and every PR improves both the code and the style guide itself
- A live example from [GloryRole PR #18](https://github.com/franklesniak/GloryRole/pull/18) showing scoring tables, rejected suggestions, and style guide updates
