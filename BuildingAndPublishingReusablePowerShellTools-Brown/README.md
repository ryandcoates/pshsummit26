# Building and Publishing Reusable PowerShell Tools

This repository contains materials from a presentation on building and publishing reusable PowerShell tools, delivered at PowerShell Summit 2026.

## Contents

- **slides/**: Contains the PowerPoint presentation file used for the conference talk.
- **demo/**: A series of PowerShell script files (`.ps1`) demonstrating key concepts step-by-step:
  - `1_SimpleCommand.ps1`: Introduction to basic PowerShell commands.
  - `2_BuildAFunction.ps1`: Creating reusable functions.
  - `3_AddParameters.ps1`: Adding parameters to functions.
  - `4_ParameterValidation.ps1`: Implementing parameter validation.
  - `5_ErrorHandling.ps1`: Handling errors in scripts.
  - `6_PipelineProcessing.ps1`: Working with pipeline input.
  - `7_AddHelp.ps1`: Adding help documentation to scripts.
  - `8_PSScriptAnalyzer.ps1`: Using PSScriptAnalyzer for code quality.
  - `9_BuildAModule.ps1`: Building a PowerShell module.
  - `10_PublishToPSGallery.ps1`: Publishing modules to the PowerShell Gallery.

These demo scripts were used live during the presentation to illustrate practical implementation of the concepts covered.
